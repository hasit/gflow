g gflow

A small shell tool for creating prefixed feature branches from a fresh base branch, cleaning up finished local branches, and installing one shell integration file for Fish, Zsh, Bash, or a generic PATH setup.

curl -fsSL https://raw.githubusercontent.com/<owner>/gflow/main/install.sh | sh

Commands

Command Use
gflow prefix team/ Save the branch prefix.
gflow new api-cleanup Create team/api-cleanup from origin/main.
gflow done team/api-cleanup
gdone team/api-cleanup 		Switch to main, pull, delete the branch, and prune origin.

Options

Variable Scope Purpose
GFLOW_INSTALL_DIR Install Change where the gflow executable is installed.
GFLOW_SHELL Install Force fish, zsh, bash, or generic PATH setup.
GFLOW_BASE_URL Install Fetch live files from a fork, mirror, or branch.
GFLOW_MAIN_BRANCH Runtime Use a base branch other than main.
GFLOW_REMOTE Runtime Use a remote other than origin.
